# AI Study Record

A personal AI learning journal built with [VitePress](https://vitepress.dev/).

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm docs:dev

# Build for production
pnpm docs:build

# Preview production build
pnpm docs:preview
```

## Deployment

This site is deployed on [Vercel](https://vercel.com). Push to `main` branch to trigger auto deployment.
